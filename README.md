# AI Workshop - OpenEdge ABL Project

An OpenEdge ABL workshop project demonstrating modern ABL patterns including:

- **Business Entity Pattern** - Layered architecture for data access
- **Factory Pattern** - Singleton management of business entities
- **Dataset-based Data Transfer** - Using ProDataSets for UI-to-entity communication

## Project Structure

- `src/business/` - Business entity classes and dataset definitions
- `src/*.w` - UI window files
- `doc/` - Architecture documentation
- `dump/` - Database schema definitions
- `.windsurf/` - AI assistant rules and workflows

## Getting Started

1. Configure OpenEdge 12.8 environment
2. Create the sports2000 database: `ant db`
3. Open the project in your IDE
